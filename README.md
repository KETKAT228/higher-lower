# 🎮 Игра "Выше-Ниже" на Python

Консольная игра с анимацией загрузки, в которой можно играть самому или дать угадывать компьютеру!

## ✨ Особенности
*  Анимация загрузки в консоли с возможностью пропуска по нажатию `ESC`.
* **Режим игрока**: угадывайте, какое число будет следующим.
*  **Режим робота**: вводите числа сами и проверяйте интуицию искусственного интеллекта.
* Защита от дурака (игра не вылетает при вводе букв).

## 🛠 Как запустить
1. Установите библиотеку `keyboard` (нужна для отслеживания клавиши ESC):
   ```bash
   pip install keyboard
   ```
2. Запустите игру:
   ```bash
   python higher_lower.py
   ```
